Расширение для браузера Chrome, которое позволяет изменять внешний вид сайта ficbook.net с помощью пользовательских CSS-стилей. Поддерживает включение/отключение отдельных модулей и кэширование стилей для повышения производительности.

## Возможности
- Включение/отключение конкретных стилей
- Вкладки во вкладке расширения
- Сохранение выбраных стилей
- Массовые действия:
  - Выбрать всё
  - Сбросить всё

## Доступные стили для выбора
Вернуть старый дизайн сайта, а именно:
  - добавить книжную обложку на фон;
  - расширеный поиск на всю страницу;
  - убрать кнопку "Вперед" в конце работы;
  - в профиле сделать больше шрифт и иконки в боковом меню;
  - увеличить размер колокольчика;
  - увеличить заголовки вкладок;
  - без промо в списках;
  - добавить подчеркивание для названий фанфиков (ссылки);
  - изменить цвет текста на более темный.

Также добавлены новые функции для удобства:
  - изменены стили для коментариев для большей читабельности;
  - скрыть прочитанные фанфики с популярного.
